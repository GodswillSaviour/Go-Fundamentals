# Go Fundamentals

My Go learning journey from scratch — concepts and projects.

## About
This repository documents my Go programming journey, covering core concepts through hands-on projects. Each folder represents a topic with a mini project demonstrating the concept.

## How to Run
Clone the repository:
    git clone https://github.com/GodswillSaviour/Go-Fundamentals.git

Navigate into any topic folder and run:
    cd 01-hello-world
    go run main.go
